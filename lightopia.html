Lightopia

In dit project heb ik een website ontwikkeld voor Lightopia, waarbij ik een vooraf ontworpen design omzette naar een volledig functionele en responsieve webpagina. De website bestaat uit meerdere secties zoals Home, Parcours, Praktisch, Galerij, en FAQ, waarbij elke pagina unieke interacties en visuele elementen bevat. Voor de dynamische onderdelen, zoals overlays voor afbeeldingen en interactieve FAQ-secties, heb ik JavaScript gebruikt. De layout is ontworpen met CSS Grid voor een nette galerijweergave en is volledig responsive om een optimale gebruikerservaring op alle apparaten te garanderen. Verder zorgde ik voor een gestructureerde mappenindeling en versiebeheer via GitHub om de voortgang en kwaliteit van het project te waarborgen.
